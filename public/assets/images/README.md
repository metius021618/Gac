# Imágenes del Sistema GAC

Este directorio contiene las imágenes utilizadas por el sistema.

## Archivos Requeridos

- `logocamb.png` - Logo principal del sistema (máximo 250px de ancho recomendado)
- `fondo.jpg` - Imagen de fondo para la página principal
- `favicon.png` - Favicon del sitio (16x16 o 32x32 px)

## Notas

Si los archivos no están presentes, el sistema seguirá funcionando pero sin las imágenes correspondientes.
Puedes copiar los archivos desde el proyecto original STREAMIN si están disponibles.
